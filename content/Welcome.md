# Welcome to Your Obsidian Vault!

This is your new Obsidian vault created at `~/workspace/agent-obsidian-vault`.

## Getting Started

- Create new notes by asking me to create them
- Search for notes using keywords
- Link notes together using `[[Wikilinks]]`
- Organize notes in folders

## What I Can Do

I can help you:
- **Read notes** - Just ask me to read a note by name
- **Search notes** - Find notes by filename or content
- **Create notes** - Create new markdown files
- **Edit notes** - Append or modify existing notes
- **List notes** - Show all notes or notes in a folder

## Example

Try asking me:
- "Create a note called 'Meeting Notes'"
- "Search for notes about 'welcome'"
- "Read my Welcome note"

Happy note-taking! 📝
